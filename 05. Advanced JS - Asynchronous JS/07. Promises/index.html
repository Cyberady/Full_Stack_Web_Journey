Promises

A Promise represents a future value that may complete successfully or fail later.

Analogy: A Job Interview

“We’ll let you know within a week.”

That statement is a promise. You don't know the result immediately. You wait for the response later.

Promise States

Pending

The promise is still waiting to complete.

Resolved / Fulfilled

The promise completed successfully.

Rejected

The promise failed or something went wrong.

Simple Promise Example

fetch("https://api.example.com/data")
  .then(response => response.json())
  .then(data => {
      console.log(data)
  })
  .catch(error => {
      console.log(error)
  })

Promises make asynchronous JavaScript easier to handle.
