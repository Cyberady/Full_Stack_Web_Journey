Data Format

Introduction to JSON

JSON is a lightweight format used to store and exchange data between applications and servers.

What is JSON?

JSON stands for JavaScript Object Notation.

It is commonly used for sending and receiving data in web applications and APIs.

JSON is easy for humans to read and easy for machines to parse.

Why Use JSON?

Readable

Simple and clean structure.

Lightweight

Transfers data quickly over networks.

Widely Used

Supported by almost every programming language.

Basic JSON Syntax

{
  "name": "Aditya",
  "age": 20,
  "skills": ["HTML", "CSS", "JavaScript"],
  "isStudent": true
}

JSON Rules

  • Data is written in key-value pairs.
  • Keys must be inside double quotes.
  • Strings use double quotes.
  • Objects are wrapped inside curly braces { }.
  • Arrays are wrapped inside square brackets [ ].

JSON is the language of modern APIs and web applications.
