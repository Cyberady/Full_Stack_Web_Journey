Understanding Requests & Responses

Learn how clients and servers communicate on the internet using requests and responses.

Request

What is a Request?

  • A request happens when a device asks for a resource from a server.
  • Resources can include HTML pages, images, JSON data, authentication tokens, and more.
  • Requests usually require an internet connection to reach the server.
Response

What is a Response?

  • A response is the reply sent back from the server.
  • The response may contain requested data such as HTML, JSON, images, or files.
  • The server can also reject requests if the client is not authorized.
